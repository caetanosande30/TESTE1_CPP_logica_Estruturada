## Descrição do Exercício

- Armazenamento e processamento de dados em vetores
- Vetores paralelos para representar entidades relacionadas
- Cálculo de médias, máximos e percentagens
- Pesquisa sequencial em vetores
- Ordenação com o algoritmo Bubble Sort
- Operações aritméticas e aplicação de descontos
- Integridade de dados em vetores paralelos

| Ficheiro | Descrição |
|---|---|
| `pergunta1.cpp` | Temperaturas médias de 7 dias — média semanal e dia mais quente |
| `pergunta2.cpp` | Sistema de stock de loja — pesquisa por código e aviso de stock crítico |
| `pergunta3.cpp` | Gestão de notas de 15 alunos — aprovados e percentagem de aprovação |
| `pergunta4.cpp` | Fatura com 5 itens — subtotais, total e desconto de 10% acima de 1000 MT |
| `pergunta5.cpp` | Ranking de 8 atletas — ordenação por tempo com Bubble Sort |
| `pergunta6.cpp` | Pesquisa de produto em supermercado — exibe preço ou "não encontrado" |
